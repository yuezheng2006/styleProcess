# Sass入门与实战

Sass is the most mature, stable, and powerful professional grade CSS extension language in the world. 打开Sass官网就可以看见这样一句话。这话一点儿都不谦虚:Sass世界上最成熟、最稳定、最强大的专业级CSS扩展语言！他如此自信想必是有各种自信的理由，下面我们就一起来了解了解Sass，看看它在吹牛还在真的如此厉害。

# css预处理器

scss是一款强化css的预处理语言，相当于一个辅助工具。它在css语法的基础上增加了变量、嵌套、混合、导入等高级功能，这些拓展令css更加强大与优雅，有助于更好地组织管理样式文件，更高效地开发项目。

# Sass特点

  • 完全兼容 CSS3
  • 在 CSS 语言基础上添加了扩展功能，比如变量(variables)、嵌套 (nesting)、混合 (mixin)
  • 对颜色和其它值进行操作的{Sass::Script::Functions 函数}
  • 函数库控制指令之类的高级功能
  • 良好的格式，可对输出格式进行定制
  • 支持 Firebug

# Sass的两种语法形式

Sass 有两种语法。

  • 第一种被称为 SCSS (Sassy CSS)，是一个 CSS3 语法的扩充版本(.scss后缀），也就是说，所有符合 CSS3 语法的样式表也都是具有相同语法意义的 SCSS 文件。
  • 第二种比较老的语法成为缩排语法（.sass后缀）， 提供了一种更简洁的 CSS 书写方式。 它不需要使用花括号，而是通过缩排的方式来表达选择符的嵌套层级，I而且也不使用分号，而是用换行符来分隔属性。

一般使用Scss多过Sass，不过二者除了上述不同外，基本在无差别
