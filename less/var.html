# 变量

如果多次使用同一值，可将其抽离为变量，更易维护：

a,
.link{
	color: black;
}
.widget{
	color: black;
	background: white;
}

可改写：

// 变量声明
@link-color: black;
@link-color-hover: darken(@link-color, 10%);

//变量使用
a,
.link{
	color: @link-color;
}
a:hover{
	color: @link-color-hover
}
.widget{
	color: #fff;
	background: @link-color;
}

# 变量插入

# 选择器

// 变量声明
@my-selector: banner;
// 变量使用
.@{my-selector} {
	font-weight: bold;
	line-height: 40px;
	margin     : 0 auto;
}

编译为：

.banner {
	font-weight: bold;
	line-height: 40px;
	margin     : 0 auto;
}

# 链接

// 变量声明
@images: "../img";
// 变量使用
body {
	color     : #444;
	background: url("@{images}/white-sand.png");
}

# 文件引用

// 变量声明
@themes: "../src/themes";
// 变量使用
@import "@{themes}/tidal-wave.less";

# 属性

@property: color;
.widget {
	@{property}           : #0ee;
	background-@{property}: #999;
}

编译为：

.widget {
	color           : #0ee;
	background-color: #999;
}

# 变量名

甚至可使用变量名命名变量：(当然实际使用中不要使用这么坑的写法哦)

@fnord: "I am fnord.";
@var: "fnord";
section{
	content: @@var;
}

编译为：

section{
	content: "I am fnord.";
}

# 懒加载（Lazy Load）

变量可先使用，再声明。

.lazy-eval-scope{
	width: @var;
}
@var: @a;
@a: 9%;

和

.lazy-eval-scope{
	width: @var;
	@a: 9%;
}
@var: @a;
@a: 100%;

都编译为：

.lazy-eval-scope{
	width: 9%;
}

如果变量多次定义，以最后一次声明为准。使用时由作用域向上查找。比如：

@var: 0;
.class {
	@var: 1;
	.brass {
		@var : 2;
		three: @var;
		@var : 3;
	}
	one : @var;
}

编译结果为：

.class{
	one: 1;
}
.class .brass{
	three: 3;
}
