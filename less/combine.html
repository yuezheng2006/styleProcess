# 合并

combine平常使用中用的不多，不过合并用起来还是很方便的。 作用：将属于同一属性的值合并在一起，比如background和transform。

# 使用逗号链接

属性名后添加符号+：

.mixin() {
	box-shadow+: inset 0 0 10px #555;
}
.myclass {
	.mixin();
	box-shadow+: inset 10px 10px 10px #ccc;
}

编译为：

.myclass {
  box-shadow: inset 0 0 10px #555, inset 10px 10px 10px #ccc;
}

# 使用空格合并

在属性名后添加符号+_：

.background-img() {
	background+_: url("test.png");
}
.background {
	background+_: #ccc;
	.background-img();
}

编译为：

.background {
  background: #ccc url("test.png");
}
