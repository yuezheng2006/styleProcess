# 循环

Less混合器可递归调用。再结合限制条件和模式匹配，可实现不同的循环结构：

.loop(@counter) when (@counter > 0) {
	.loop(@counter - 1);
	width: (10px * @counter);
}
div {
	.loop(5);
}

编译为：

div {
  width: 10px;
  width: 20px;
  width: 30px;
  width: 40px;
  width: 50px;
}

再比如，生成栅格系统：

.generate-columns(@n, @i:1) when (@i <= @n) {
	.column-@{i} {
		width: @i / @n * 100%;
	}
	.generate-columns(@n, @i+1);
}

// 调用 n=4
.generate-columns(4);

编译为：

.column-1 {
	width: 25%;
}
.column-2 {
	width: 50%;
}
.column-3 {
	width: 75%;
}
.column-4 {
	width: 100%;
}

// 调用 n=2
.generate-columns(2);

编译为：

.column-1 {
	width: 50%;
}
.column-2 {
	width: 100%;
}
